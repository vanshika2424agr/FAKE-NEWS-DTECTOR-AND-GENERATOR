# Fake News Detector 

## Overview
This project uses NLP and Machine Learning to detect fake news articles.

## Technologies Used
- Python
- Scikit-Learn
- TF-IDF Vectorization
- Logistic Regression
- Transformers (GPT-2)
- Gradio
- Google Colab

## Features
- Fake News Detection
- Confidence Score
- Fictional News Generation
- Interactive User Interface

## Model Accuracy
99.08%
